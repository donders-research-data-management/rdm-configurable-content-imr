Share data

1. How do I share data with collaborators at the Donders/RU/UMC?

If you are the manager of a collection you can easily share data with local collaborators for all collection types. Simply choose Edit -> Add Contributor/Viewer and then select from the registered users. If your colleague is not yet in the list, instruct him/her to register an account on the Donders Repository using his/her u- or z-number as described here.

2. How do I share data with collaborators in the rest of the world?

Sharing data with collaborators outside the local centers requires these collaborators to identify themselves through one of multiple mechanisms as described here. The initiative for sharing the data comes from the collaborator as described here. If a Data Acquisition Collection should be shared with outside collaborators, you need provide the collaborator with the URL to the repository based on the reserved persistent identifier, since the DAC will not be public.

3. How do I share data with external reviewers?

The Donders Repository includes the possibility of sharing data from a Data Sharing Collection with external reviewers (i.e. before publication) in a way that protects the anonymity of the reviewers. For details on this see the manual on data sharing.

4. Can I share my data through an external repository?

Data collected within the Donders Institute should in the first place be shared through the Donders Repository.

You may, however, want to add your data to a repository that is more common in your field, or a journal editor may oblige you to provide your data in an external repository.

This is allowed only if the external repository handles data in compliance with privacy legislation and RU RDM policy:

  • Data ownership must remain in hands of the data creator (RU) only
  • The repository must allow (custom) Data Use Agreements that fit the type of research data, see this FAQ.

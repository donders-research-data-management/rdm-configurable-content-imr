Request access

You can find and retrieve shared data in the Donders Research Data Repository through a persistent identifier (see this FAQ: What is a persistent identifier?) that is specific to the data collection. You can obtain these persistent identifiers in two ways:

  1. directly from the author(s) of the data;
  2. from a publication that is linked to the data.

For example, say you are interested in the data used in the paper by van der Meij, van Ede and Maris (2016, PLos ONE).You can find the persistent identifier linked to the data on the publication’s webpage.

(http://journals.plos.org/plosone/article?id=10.1371/journal.pone.0154881)

In the Donders Research Data Repository, shared data can be found in a Data Sharing Collection(DSC). Click on the link, which will guide you to DSC’s page in the repository.

Without logging in, you can only see limited information of the data collection. When you click the “Request access” button, your first need to log in.

To log in, follow the instructions in 1.1 Login in Portal.

Go to the shared collections page. On the data sharing collection page, now click the “Request access” button.

Then, you will see a Data Use Agreement (see this FAQ: what is a Data Use Agreement?) for this particular DSC.

Read it carefully and tick the box at the end of the page if you agree with the Data Use Agreement. Then click on the blue bar to get access to the data.

If you follow these steps, you will be automatically added as a viewer of this collection. As a viewer, you can view all collection details and you can download the data from the collection (see this FAQ: What are the differences between the roles in a collection?).
